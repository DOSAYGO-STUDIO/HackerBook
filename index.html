Hacker Book

Community, All the HN Belong to You! 2006 - 2025 FOREVER

Static, offline-friendly Hacker News archive. Everything runs in your browser via SQLite WASM; only the shards you need are fetched.

Live Demo GitHub Repo Open Local Viewer

Quick start: browse the archive locally

Serve the docs/ folder with any static file server:

# with npm
npx serve docs

# or with Python
python3 -m http.server 8000 --directory docs

# then open
http://localhost:8000

Use the date picker to time-warp; queries run entirely client-side.

What’s inside

  • docs/static-shards/: gzipped SQLite shards of items + comments
  • docs/static-user-stats-shards/: gzipped SQLite shards with per-user stats and monthly activity
  • Manifests/indexes (all gzipped): static-manifest.json.gz, archive-index.json.gz, cross-shard-index.bin.gz, static-user-stats-manifest.json.gz
  • Everything is static; no backend required.

Regenerate user stats shards (optional)

npm install
node toool/s/build-user-stats.mjs --gzip --target-mb 15

Then serve docs/ as above.

Notes

  • Optimized for modern browsers (Chrome, Firefox, Safari). Uses DecompressionStream when available, falls back to pako.
  • Mobile layout is locked to the viewport; works offline once needed shards are cached.
